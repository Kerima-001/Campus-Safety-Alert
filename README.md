# Campus Safety Alert System

### Real-Time Emergency Response & Student Safety Platform

A modern campus safety application designed to improve emergency communication, incident reporting, situational awareness, and response coordination across university campuses.

Built with **Python, FastAPI, Flutter, PostgreSQL, Firebase Cloud Messaging, and Location Services**.

---

#  Overview

Campus safety is a critical component of student success and well-being. Universities often face challenges in rapidly communicating emergencies, tracking incidents, and providing students with real-time safety information.

The **Campus Safety Alert System** provides a centralized platform that enables students, faculty, and campus security personnel to report incidents, receive emergency alerts, share live locations, and coordinate responses during critical situations.

The platform combines mobile technology, geolocation services, push notifications, and real-time communication tools to enhance campus security and emergency preparedness.
